# HTML Profile v1.0

## Overview
An HTML profile website. Think of it as a social media profile that you own and can fully customize.

## Files
- index.html: Your HTML profile page
- style.css: Contains all the styling for the profile layout and elements
- avatar.jpg: Profile image used throughout the site

## Getting Started
1. To view this project, simply open the index.html file in your web browser
2. To customize with your own information, edit the content in index.html with your personal details
3. To change the profile image, replace the avatar.jpg file with your own image
4. Customize the style.css file to change the layout and design of the profile

## Features
- Energy orb
- Responsive layout for mobile
- Simple navigation with webring links
